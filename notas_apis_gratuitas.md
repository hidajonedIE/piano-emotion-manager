# Notas de Investigación: APIs de IA Gratuitas

## 1. Google Gemini API - Plan Gratuito

### Modelos con Plan Gratuito:
- **Gemini 3 Flash Preview**: Input/Output GRATIS
- **Gemini 2.5 Pro**: Input/Output GRATIS
- **Gemini 2.5 Flash**: Input/Output GRATIS (+ Grounding con Google Search hasta 500 RPD)
- **Gemini 2.5 Flash-Lite**: Input/Output GRATIS
- **Gemini 2.0 Flash**: Input/Output GRATIS
- **Gemma 3 / Gemma 3n**: Modelos open-source gratuitos

### Limitaciones del Plan Gratuito:
- Acceso limitado a ciertos modelos (Gemini 3 Pro NO disponible gratis)
- Rate limits más bajos que el plan de pago
- **El contenido SE USA para mejorar los productos de Google**
- Sin acceso a Context caching
- Sin Batch API

### Precios Plan de Pago (referencia):
- Gemini 2.5 Flash: $0.30/1M tokens entrada, $2.50/1M tokens salida
- Gemini 2.5 Pro: $1.25/1M tokens entrada, $10.00/1M tokens salida
