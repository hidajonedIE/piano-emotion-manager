# Glosario Maestro de Términos Clave


## Introducción


Este glosario define las traducciones estándar para términos clave de Piano Emotion Manager.

Debe ser utilizado como referencia para mantener consistencia terminológica en todas las traducciones.


## Citas y Calendario


### `appointment`


**Contexto:** Cita de servicio programada


**Traducciones:**


| Idioma | Traducción |

|---|---|

| Danés | `aftale` |

| Alemán | `Termin` |

| Inglés | `appointment` |

| Español | `cita` |

| Francés | `rendez-vous` |

| Italiano | `appuntamento` |

| Noruego | `avtale` |

| Portugués | `agendamento` |

| Sueco | `möte` |


**Notas:** Preferir 'cita' sobre 'servicio' para claridad.
