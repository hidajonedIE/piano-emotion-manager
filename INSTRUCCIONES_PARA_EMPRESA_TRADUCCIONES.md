## 1. Descripción del Proyecto

Piano Emotion Manager es una aplicación SaaS completa de gestión de negocio para técnicos de piano. La aplicación incluye módulos para:

- Gestión de clientes y pianos
- Programación de citas y calendario
- Facturación y pagos
- Inventario y proveedores
- Análisis y reportes
- Geolocalización y optimización de rutas

## 3. Proceso de Revisión Recomendado

### Fase 1: Revisión Terminológica (Prioridad Alta)

**Objetivo:** Asegurar que los términos clave se traducen consistentemente.

**Pasos:**
1. Abrir el archivo XLSX
2. Consultar el Glosario Maestro para cada término clave
3. Verificar que las traducciones coincidan con el glosario
4. Corregir cualquier variación no autorizada
5. Documentar cambios realizados

**Términos Críticos a Revisar:**
- `invoice` / `factura` - Implicaciones legales
- `appointment` / `cita` - Concepto central
- `client` / `cliente` - Definición de usuario
- `piano` / `pianoforte` - Producto principal
- `tuning` / `afinación` - Servicio principal
- `maintenance` / `mantenimiento` - Servicio diferenciado

### Fase 2: Revisión Lingüística (Prioridad Alta)

**Objetivo:** Asegurar calidad gramatical y naturalidad.

**Pasos:**
1. Revisar gramática y ortografía en cada idioma
2. Verificar concordancia de género y número
3. Asegurar naturalidad del lenguaje
4. Corregir errores de puntuación

**Consideraciones por Idioma:**

- **Alemán:** Verificar capitalización de sustantivos, géneros gramaticales
- **Danés:** Revisar palabras compuestas, vocabulario técnico
- **Noruego:** Diferenciación clara del sueco, vocabulario especializado
- **Sueco:** Diferenciación clara del noruego, palabras compuestas
- **Francés:** Variantes regionales (Francia vs otros países francófonos)
- **Portugués:** Variantes regionales (Portugal vs Brasil)
- **Italiano:** Vocabulario técnico especializado
- **Español:** Variantes regionales (España vs América Latina)

### Fase 3: Revisión Contextual (Prioridad Media)

**Objetivo:** Asegurar que las traducciones tienen sentido en contexto.

**Pasos:**
1. Revisar cada traducción en contexto
2. Verificar que el significado se preserva
3. Asegurar que es apropiado para el público objetivo
4. Corregir si es necesario

**Contextos a Considerar:**
- Interfaz de usuario (botones, etiquetas, mensajes)
- Mensajes de error y notificaciones
- Textos de ayuda y documentación
- Términos de negocio y facturación

### Fase 4: Validación Final (Prioridad Media)

**Objetivo:** Asegurar consistencia y calidad general.

**Pasos:**
1. Revisar el archivo completo una vez más
2. Verificar que no hay traducciones faltantes
3. Asegurar que no hay duplicados o inconsistencias
4. Generar reporte final de cambios

## 4. Criterios de Calidad

### Aceptable

- ✓ Traducción gramaticalmente correcta
- ✓ Consistente con el glosario maestro
- ✓ Natural y apropiada para el contexto
- ✓ Sin errores de ortografía
- ✓ Mantiene el significado original

### No Aceptable

- ✗ Errores gramaticales o de ortografía
- ✗ Inconsistente con el glosario maestro
- ✗ Suena poco natural o forzado
- ✗ Pierde el significado original
- ✗ Inapropiado para el contexto

## 6. Consideraciones Especiales

### Términos Técnicos

Algunos términos son específicos del sector de pianos y servicios musicales:

- **Piano Types:** Grand, Baby Grand, Concert Grand, Upright, Spinet, Console, Digital
- **Services:** Tuning, Maintenance, Repair, Regulation, Voicing
- **Piano Parts:** Strings, Soundboard, Action, Pedals, Keys

Recomendamos consultar con expertos en música/pianos si es necesario.

### Regulaciones Locales

Algunos términos tienen implicaciones legales o fiscales:

- **Invoice/Factura:** Debe cumplir con regulaciones fiscales locales
- **Tax ID:** Nomenclatura específica por país (NIF, CIF, VAT, etc.)
- **Payment Terms:** Pueden variar por país

Recomendamos validar con expertos legales/fiscales si es necesario.

### Variantes Regionales

Para idiomas con múltiples variantes, recomendamos:

- **Portugués:** Especificar si es Portugal (PT-PT) o Brasil (PT-BR)
- **Español:** Especificar si es España o América Latina
- **Francés:** Especificar si es Francia o otros países francófonos
