## Requisitos del Sistema

Piano Emotion Manager funciona en la mayoría de dispositivos modernos. A continuación se detallan los requisitos mínimos para cada plataforma.

### Dispositivos Móviles

| Plataforma | Versión Mínima | Espacio Necesario |
|------------|----------------|-------------------|
| **iPhone** | iOS 14.0 o superior | 50 MB |
| **iPad** | iPadOS 14.0 o superior | 50 MB |
| **Android** | Android 8.0 o superior | 50 MB |

### Ordenador

| Navegador | Versión Recomendada |
|-----------|---------------------|
| **Google Chrome** | Versión 90 o superior |
| **Safari** | Versión 14 o superior |
| **Firefox** | Versión 88 o superior |
| **Microsoft Edge** | Versión 90 o superior |

## Instalación en iPhone

### Paso 1: Acceder a la App Store

Localiza el icono de la **App Store** en la pantalla de inicio de tu iPhone y tócalo para abrirla.

### Paso 2: Buscar la Aplicación

En la pestaña **Buscar** (icono de lupa en la parte inferior), escribe "Piano Emotion Manager" en el campo de búsqueda y pulsa Buscar.

### Paso 3: Descargar e Instalar

1. Localiza **Piano Emotion Manager** en los resultados de búsqueda (icono del piano de cola)
2. Toca el botón **Obtener**
3. Confirma la descarga con Face ID, Touch ID o tu contraseña de Apple ID
4. Espera a que se complete la instalación

### Paso 4: Abrir la Aplicación

Una vez instalada, encontrarás el icono de Piano Emotion Manager en tu pantalla de inicio. Tócalo para abrir la aplicación y comenzar a usarla.

---

## Instalación en Android

### Paso 1: Acceder a Google Play Store

Localiza el icono de **Play Store** en tu dispositivo Android y tócalo para abrirlo.

### Paso 2: Buscar la Aplicación

Toca la barra de búsqueda en la parte superior y escribe "Piano Emotion Manager". Pulsa el icono de búsqueda o la tecla Enter.

### Paso 3: Descargar e Instalar

1. Localiza **Piano Emotion Manager** en los resultados (icono del piano de cola)
2. Toca el botón **Instalar**
3. Acepta los permisos solicitados si aparece alguna ventana
4. Espera a que se complete la descarga e instalación

### Paso 4: Abrir la Aplicación

Puedes abrir la aplicación directamente desde Play Store tocando **Abrir**, o buscar el icono de Piano Emotion Manager en tu cajón de aplicaciones.

## Acceso desde Ordenador

Piano Emotion Manager está disponible como aplicación web, lo que significa que puedes acceder desde cualquier ordenador con conexión a internet sin necesidad de instalar nada.

### Acceso Rápido

1. Abre tu navegador web preferido (Chrome, Safari, Firefox o Edge)
2. Escribe en la barra de direcciones: **pianoemotion.es**
3. Pulsa Enter y la aplicación se cargará automáticamente
4. Ya puedes empezar a usar todas las funciones

### Guardar como Favorito

Para acceder más rápidamente en el futuro, te recomendamos guardar la página como favorito:

**En Chrome/Edge:** Pulsa Ctrl+D (Windows) o Cmd+D (Mac) y confirma.

**En Safari:** Pulsa Cmd+D y selecciona la carpeta de favoritos.

**En Firefox:** Pulsa Ctrl+D (Windows) o Cmd+D (Mac) y confirma.

### Instalar como Aplicación de Escritorio (Opcional)

Si prefieres tener Piano Emotion Manager como una aplicación independiente en tu ordenador, puedes instalarla como aplicación web progresiva (PWA).

**En Google Chrome:**
1. Abre pianoemotion.es en Chrome
2. Haz clic en el icono de instalación que aparece en la barra de direcciones (un ordenador con una flecha hacia abajo)
3. Confirma la instalación haciendo clic en "Instalar"
4. La aplicación se añadirá a tu escritorio y menú de inicio

**En Microsoft Edge:**
1. Abre pianoemotion.es en Edge
2. Haz clic en los tres puntos del menú (esquina superior derecha)
3. Selecciona "Aplicaciones" → "Instalar este sitio como aplicación"
4. Confirma la instalación

## Primer Inicio

Al abrir Piano Emotion Manager por primera vez, la aplicación te guiará a través de una breve configuración inicial.

### Pantalla de Bienvenida

La primera vez que abras la aplicación, verás el Dashboard principal con las siguientes secciones:

| Sección | Descripción |
|---------|-------------|
| **Acciones Rápidas** | Botones para crear clientes, pianos y servicios rápidamente |
| **Este Mes** | Resumen de tu actividad del mes actual |
| **Módulos** | Acceso a todas las funciones de la aplicación |
| **Últimos Servicios** | Historial de tus servicios más recientes |

### Configuración Recomendada

Antes de empezar a registrar clientes y servicios, te recomendamos completar estos pasos:

1. **Configura tus datos fiscales**: Accede a Módulos → Datos Fiscales e introduce tu información de facturación (nombre/razón social, NIF/CIF, dirección, etc.)

2. **Personaliza tus tarifas**: En Módulos → Tarifas, define los precios de tus servicios habituales (afinación, regulación, reparaciones, etc.)

3. **Explora la ayuda**: En Módulos → Configuración → Ayuda encontrarás instrucciones detalladas de cada función

### Permisos en Dispositivos Móviles

La aplicación puede solicitar algunos permisos para funcionar correctamente:

| Permiso | Para qué se usa | ¿Es obligatorio? |
|---------|-----------------|------------------|
| **Notificaciones** | Recordatorios de citas programadas | Recomendado |
| **Almacenamiento** | Guardar copias de seguridad | Recomendado |

Puedes gestionar estos permisos en cualquier momento desde los ajustes de tu dispositivo.

## Preguntas Frecuentes

### ¿La aplicación es gratuita?

Sí, Piano Emotion Manager es completamente gratuita. No hay suscripciones, compras dentro de la app ni funciones bloqueadas.

### ¿Mis datos están seguros?

Tus datos se almacenan de forma segura en tu dispositivo. Además, puedes crear copias de seguridad manuales desde la sección de Configuración para proteger tu información.

### ¿Puedo usar la app sin conexión a internet?

Sí, la aplicación funciona completamente sin conexión. Todos tus datos se guardan localmente en tu dispositivo.

### ¿Cómo transfiero mis datos a un nuevo dispositivo?

Utiliza la función de Copia de Seguridad:
1. En tu dispositivo antiguo: Configuración → Copia de Seguridad → Exportar
2. Guarda el archivo generado
3. En tu nuevo dispositivo: Configuración → Copia de Seguridad → Importar
4. Selecciona el archivo guardado

### ¿La app funciona en tablet?

Sí, Piano Emotion Manager está optimizada para funcionar tanto en teléfonos como en tablets, adaptando su interfaz al tamaño de pantalla.

### ¿Puedo usar la misma cuenta en varios dispositivos?

Actualmente, los datos se almacenan localmente en cada dispositivo. Para sincronizar datos entre dispositivos, utiliza la función de exportar/importar copias de seguridad.

### ¿Cómo actualizo la aplicación?

**En iPhone:** Las actualizaciones se descargan automáticamente si tienes activadas las actualizaciones automáticas. También puedes actualizar manualmente desde la App Store.

**En Android:** Las actualizaciones se gestionan desde Google Play Store. Puedes activar las actualizaciones automáticas o actualizar manualmente.

**En ordenador:** La versión web se actualiza automáticamente cada vez que accedes a pianoemotion.es.
