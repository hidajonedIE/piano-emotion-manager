# Guía de Instalación de Piano Emotion Manager

Piano Emotion Manager es una aplicación profesional diseñada para técnicos de piano que permite gestionar clientes, pianos, servicios, inventario y facturación desde cualquier dispositivo. Esta guía te ayudará a instalar y acceder a la aplicación en tu móvil, tablet u ordenador.

---

## Instalación en iPhone y iPad (iOS)

La aplicación se puede instalar como una **Progressive Web App (PWA)** directamente desde Safari, sin necesidad de descargarla desde la App Store.

### Pasos de instalación

1. **Abre Safari** en tu iPhone o iPad. Es importante usar Safari, ya que otros navegadores como Chrome no permiten instalar PWAs en iOS.

2. **Accede a la aplicación** navegando a la URL proporcionada por tu administrador o la dirección oficial de Piano Emotion Manager.

3. **Pulsa el botón de compartir** (el icono de cuadrado con flecha hacia arriba) situado en la barra inferior de Safari.

4. **Desplázate hacia abajo** en el menú que aparece y selecciona **"Añadir a pantalla de inicio"**.

5. **Personaliza el nombre** si lo deseas (por defecto aparecerá "Piano Emotion Manager") y pulsa **"Añadir"** en la esquina superior derecha.

6. **Accede desde tu pantalla de inicio**. La aplicación aparecerá como un icono más junto a tus otras apps, y se abrirá a pantalla completa sin barras del navegador.

### Requisitos

| Requisito | Especificación |
|-----------|----------------|
| Sistema operativo | iOS 14.0 o superior |
| Navegador | Safari (obligatorio para la instalación) |
| Conexión | Internet para sincronización |
| Almacenamiento | Aproximadamente 50 MB |

## Instalación en Android (Móvil y Tablet)

En dispositivos Android, la instalación es aún más sencilla gracias al soporte nativo de PWAs en Chrome y otros navegadores basados en Chromium.

### Pasos de instalación

1. **Abre Google Chrome** (o cualquier navegador compatible como Edge, Brave o Samsung Internet) en tu dispositivo Android.

2. **Navega a la URL** de Piano Emotion Manager proporcionada por tu administrador.

3. **Espera unos segundos** hasta que aparezca un banner en la parte inferior de la pantalla con el mensaje **"Añadir Piano Emotion Manager a la pantalla de inicio"**. Si no aparece automáticamente, continúa con el paso 4.

4. **Pulsa el menú de tres puntos** en la esquina superior derecha del navegador.

5. **Selecciona "Instalar aplicación"** o **"Añadir a pantalla de inicio"** según la versión de tu navegador.

6. **Confirma la instalación** pulsando "Instalar" en el diálogo que aparece.

7. **Accede desde tu pantalla de inicio** o cajón de aplicaciones. La app se comportará como una aplicación nativa con su propio icono y se abrirá a pantalla completa.

### Requisitos

| Requisito | Especificación |
|-----------|----------------|
| Sistema operativo | Android 8.0 o superior |
| Navegador | Chrome 80+, Edge, Brave o Samsung Internet |
| Conexión | Internet para sincronización |
| Almacenamiento | Aproximadamente 50 MB |

## Uso en Ordenador (Windows, Mac y Linux)

Piano Emotion Manager funciona perfectamente en cualquier ordenador a través del navegador web, y también puede instalarse como aplicación de escritorio.

### Opción 1: Usar directamente en el navegador

1. **Abre tu navegador preferido** (Chrome, Edge, Firefox, Safari o Brave).

2. **Navega a la URL** de Piano Emotion Manager.

3. **Inicia sesión** si es necesario y comienza a usar la aplicación directamente.

4. **Añade a favoritos** para acceder rápidamente en el futuro.

### Opción 2: Instalar como aplicación de escritorio

Esta opción está disponible en Chrome, Edge y otros navegadores basados en Chromium, y ofrece una experiencia más similar a una aplicación nativa.

**En Google Chrome:**

1. Navega a la URL de Piano Emotion Manager.
2. Haz clic en el icono de **instalación** (un ordenador con flecha hacia abajo) que aparece en la barra de direcciones, o ve a **Menú (⋮) → Instalar Piano Emotion Manager**.
3. Confirma la instalación en el diálogo que aparece.
4. La aplicación se añadirá a tu escritorio y menú de inicio.

**En Microsoft Edge:**

1. Navega a la URL de Piano Emotion Manager.
2. Haz clic en **Menú (···) → Aplicaciones → Instalar este sitio como aplicación**.
3. Personaliza el nombre si lo deseas y confirma.
4. La aplicación aparecerá en tu menú de inicio y podrás anclarla a la barra de tareas.

### Requisitos

| Requisito | Especificación |
|-----------|----------------|
| Sistema operativo | Windows 10+, macOS 10.14+, o cualquier distribución Linux moderna |
| Navegador | Chrome 80+, Edge 80+, Firefox 90+, Safari 14+ |
| Resolución | Mínimo 1280×720 (recomendado 1920×1080) |
| Conexión | Internet para sincronización |

## Solución de Problemas Comunes

### La aplicación no se instala en iOS

Asegúrate de estar usando **Safari**. Otros navegadores como Chrome o Firefox en iOS no permiten instalar PWAs. Si el problema persiste, verifica que tu dispositivo tenga iOS 14.0 o superior.

### No aparece la opción de instalar en Android

Algunos navegadores requieren que visites la página varias veces antes de mostrar el banner de instalación. Alternativamente, accede al menú del navegador y busca la opción "Instalar aplicación" o "Añadir a pantalla de inicio".

### La aplicación no carga correctamente

Verifica tu conexión a Internet. Si el problema persiste, intenta limpiar la caché del navegador o reinstalar la aplicación eliminándola de la pantalla de inicio y volviéndola a añadir.

### Los datos no se sincronizan

Piano Emotion Manager almacena datos localmente para funcionar sin conexión. La sincronización se realiza automáticamente cuando hay conexión a Internet. Si los datos no aparecen en otro dispositivo, asegúrate de que ambos dispositivos tienen conexión y están usando la misma cuenta.
